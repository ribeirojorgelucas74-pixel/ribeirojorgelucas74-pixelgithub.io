# Sistema de Gestão de Bolsistas - PROGRAD/UENF

Este projeto consiste em um ecossistema digital desenvolvido para a **Pró-Reitoria de Graduação** da UENF.

### 🛠️ Tecnologias e Metodologias
- **Google Apps Script:** Lógica de back-end para integração com o Google Sheets.
- **HTML/CSS/JS:** Dashboard dinâmico para filtragem e visualização de dados.
- [cite_start]**Gestão de Prazos:** Controle automatizado de vigências contratuais de 12 meses[cite: 12, 86].
- [cite_start]**Segurança:** Sistema adaptado para lidar com Termos de Outorga e Aditivos com assinaturas eletrônicas[cite: 5, 43, 80, 99].

### 📄 Contexto Institucional
[cite_start]O sistema foi desenvolvido para atender às necessidades do Edital de Apoio ao Ensino, operando sob a carga horária de 20 horas semanais e seguindo as normas da Resolução nº 02/2008 da universidade[cite: 14, 61, 92].
